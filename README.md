# E-Commerce Backend API (Spring Boot)

## Project Overview
This project is a scalable backend REST API for an e-commerce platform built using Java Spring Boot.
It focuses on security, clean architecture, and real-world business use cases.

## Features
- JWT-based authentication and authorization
- Role-based access (Admin / Customer)
- Product and category management
- Order creation and order tracking
- Pagination, validation, and global exception handling
- Secure password encryption

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- MySQL / PostgreSQL
- REST APIs

## Architecture
- Controller layer for REST endpoints
- Service layer for business logic
- Repository layer for database access
- Centralized exception handling

## Purpose
This project demonstrates secure backend API development,
clean code practices, and real-world e-commerce backend workflows.
